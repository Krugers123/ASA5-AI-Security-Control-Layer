# ASA 5 Public

ASA 5 Public is the public-safe documentation track for ASA 5.

Its purpose is to expose only the materials that are suitable for:

- public positioning
- architecture communication
- partner-safe high-level review
- non-sensitive documentation

This repository is not the full project.

It is the controlled public layer for:

- selected diagrams
- public architecture summaries
- safe product framing
- non-sensitive documentation for ASA 5

## Scope

ASA 5 is being developed as:

`AI Security Control Layer`

The public track should communicate:

- external AI security signaling
- trajectory integrity
- runtime risk awareness
- operator / governance integration

without exposing the private implementation layer.

## Rule

If a document contains:

- sensitive implementation details
- private strategy
- private architecture decisions
- restricted integration logic
- anything that should remain partner-only or private

it does not belong here.
