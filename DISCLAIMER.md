# Disclaimer

ASA5 / ASC is an implemented local prototype, an architectural research direction, and a public-safe runtime security and governance-observability framework.

This public repository does not represent a certified safety-critical deployment system by itself.

The current system is suitable for controlled demonstration and scoped partner evaluation. It is not independently validated, statistically calibrated for production risk prediction, or represented as a generally production-ready system.

Any production use would require:

- formal validation
- domain-specific testing
- threat modeling
- policy integration
- access control and authentication hardening
- security review
- human governance
- operational monitoring

ASA5 / ASC does not claim ISO 26262, UL 4600, or equivalent functional-safety certification or safety case coverage.

This repository does not expose private scoring logic, thresholds, calibration data, proprietary detector internals, or sensitive deployment details.
